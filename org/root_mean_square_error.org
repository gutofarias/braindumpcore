This error is given by:
\[
RSME = \sqrt \frac{\sum^{N}_{1} [e(i) - \hat{e}(i)]^2}{N}
\]

Where:
- $N$ :: Data number
- $e(i)$ :: Error at datum i
- $\hat{e}(i)$ :: Predicted error at datum i
