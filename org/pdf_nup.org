#+title: PDF Nup

- tags :: [[id:8742274c-9645-42a7-b744-48d5720e280c][Pdf Tools]]

This puts several pages of a PDF together. ~2x1~ is a particularly
common combination.

#+begin_src bash
alias p2x1="pdfnup --nup 2x1 --landscape --suffix '2x1' --batch "
#+end_src
